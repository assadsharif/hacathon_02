# Productive Flow Theme

**Description:** A modern, energizing theme perfect for productivity apps, AI assistants, and task management tools

## Color Palette

### Primary Blue
- **Hex:** #3b82f6
- **Usage:** Primary actions, headers, links, active states
- **Represents:** Trust, productivity, focus

### Success Green
- **Hex:** #10b981
- **Usage:** Completed tasks, success messages, positive feedback
- **Represents:** Achievement, completion, growth

### Accent Purple
- **Hex:** #8b5cf6
- **Usage:** AI features, premium elements, highlights
- **Represents:** Innovation, intelligence, creativity

### Neutral Gray
- **Hex:** #f3f4f6
- **Usage:** Backgrounds, cards, subtle elements
- **Represents:** Clarity, minimalism, focus

### Supporting Colors

#### Dark Text
- **Hex:** #1f2937
- **Usage:** Primary text, headings

#### Medium Gray
- **Hex:** #6b7280
- **Usage:** Secondary text, descriptions

#### White
- **Hex:** #ffffff
- **Usage:** Card backgrounds, primary background

#### Danger Red
- **Hex:** #ef4444
- **Usage:** Delete actions, errors, warnings

## Typography

### Headers
- **Font:** DejaVuSans Bold
- **Characteristics:** Clean, modern, highly readable
- **Usage:** Page titles, section headers, card titles

### Body Text
- **Font:** DejaVuSans
- **Characteristics:** Professional, contemporary, excellent screen readability
- **Usage:** Descriptions, body content, UI text

## Visual Hierarchy

### Primary Elements
- **Color:** Primary Blue (#3b82f6)
- **Font:** DejaVuSans Bold
- **Example:** Main navigation, primary buttons, page titles

### Secondary Elements
- **Color:** Medium Gray (#6b7280)
- **Font:** DejaVuSans
- **Example:** Descriptions, metadata, timestamps

### Interactive Elements
- **Default:** Primary Blue (#3b82f6)
- **Hover:** Darker Blue (#2563eb)

- **Disabled:** Neutral Gray (#f3f4f6)

## Theme Philosophy

**Productive Flow** embodies:
- **Clarity:** Clean, uncluttered design that keeps users focused
- **Energy:** Vibrant blues and greens that motivate action
- **Intelligence:** Purple accents that signal AI-powered features
- **Trust:** Professional color choices that inspire confidence
- **Achievement:** Visual reinforcement of task completion

This theme creates an environment where users feel:
- Focused and organized
- Motivated to complete tasks
- Confident in the tool's capabilities
- Energized by visual feedback
- Supported by intelligent features
